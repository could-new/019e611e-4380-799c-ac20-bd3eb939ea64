# Fanfic Writer

A dedicated space for authors to draft, organize, and manage their fanfictions.

## Features
- **Dashboard:** Keep track of all your ongoing fanfictions.
- **Story Organization:** Add a title, fandom tag, and synopsis.
- **Chapter Management:** Create, edit, and arrange chapters within a story.
- **Distraction-Free Editor:** A clean, focused writing interface that auto-saves your progress.

## Tech Stack
- Flutter
- Material 3 Design
- In-memory data management (can be swapped for SQLite or Supabase)

## Getting Started

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
